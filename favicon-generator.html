GiveBack Hub - Favicon Setup Instructions

Step 1: Prepare Your Logo

Use your existing public/give back.png file as the source image.

For best results, make sure it's square (same width and height) and at least 512x512 pixels.

Step 2: Generate Favicon Files (Choose one method)

Method A: Online Generator (Recommended)

  1. Go to favicon.io/favicon-converter/
  2. Upload your give back.png file
  3. Download the generated favicon package
  4. Extract and copy the files to your project

Method B: RealFaviconGenerator (Advanced)

  1. Go to realfavicongenerator.net
  2. Upload your logo and customize settings
  3. Generate and download the package

Step 3: Copy Files to Your Project

After generating, copy these files to your project:

  • favicon.ico → public/favicon.ico
  • icon-16x16.png → public/icons/icon-16x16.png
  • icon-32x32.png → public/icons/icon-32x32.png
  • apple-touch-icon.png → public/icons/apple-touch-icon.png
  • All other icon sizes → public/icons/

Step 4: Verify Setup

After copying the files:

  1. Start your development server: npm run dev
  2. Open your browser and check if the favicon appears in the tab
  3. Test on mobile devices for app icons
